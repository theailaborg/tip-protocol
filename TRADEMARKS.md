# Trademark Guidelines

## Owned Marks

The following are trademarks of The AI Lab Intelligence Unobscured, Inc.
("The AI Lab") and are not granted by the TIPCL-1.0 software license:

- **TIP™**: TIP Protocol, Trust Identity Protocol
- **AI Trust ID™**: The personal trust credential system
- **AI Trust Registry™**: The credential issuance registry
- **The Global Seal of Trust™**: The network identity
- **AI Trust ID Seal** (visual design): The circular gold-ring credential badge

## What You CAN Do Without a Trademark License

These uses are permitted for anyone implementing TIP Protocol:

✓ State that your product "implements TIP Protocol"  
✓ State that your product "is compatible with TIP Protocol"  
✓ State that your product "is built on TIP Protocol"  
✓ Use the phrase "TIP Protocol" in documentation, technical writing, and code comments  
✓ Display the **TIP Powered Mark** (the open compliance badge) if you implement the protocol: see the TIP Powered Mark Display Guidelines  
✓ Display an AI Trust ID Seal that The AI Lab's registry has issued to a user: you are displaying their credential, not issuing it  

## What Requires a Trademark License

The following require written permission from The AI Lab:

✗ Using "TIP™", "TIP Protocol™", or confusingly similar marks in your product name  
✗ Using "AI Trust ID™" in your product name, service name, or domain  
✗ Using "AI Trust Registry™" to describe your credential system  
✗ Displaying the AI Trust ID Seal visual design as your own product's badge  
✗ Creating a badge or credential system that is visually confusingly similar to the AI Trust ID Seal  
✗ Using any of the marks in advertising in a way that implies endorsement by The AI Lab  

## The TIP Powered Mark (Open Compliance Badge)

The **TIP Powered Mark** is a separate open compliance badge. Any developer
or platform that correctly implements the TIP Protocol specification may
display the TIP Powered Mark, subject to these rules:

✓ Display on any platform implementing TIP Protocol  
✓ Use in developer documentation and SDKs  
✓ Show on "About" or "Trust" pages  
✓ Use at any size from 40px to 400px  
✓ Animate with CSS (scale, fade) without modifying the mark  

✗ Never substitute for the AI Trust ID™ Seal: they are different objects  
✗ Never modify the mark or add third-party branding inside the ring  
✗ Never claim the mark implies personal identity verification  
✗ Never display if you have not implemented TIP Protocol  

## Naming Your Product

If you build a product on TIP Protocol, name it something original. Examples:

**Acceptable:**  
"VerifyMe: Built on TIP Protocol"  
"NewsVerify (powered by TIP Protocol)"  
"GovID: TIP-Compatible Digital Identity"  

**Not acceptable:**  
"TIP Identity Platform"  
"MyTIP Verifier"  
"AI Trust System" (confusingly similar to AI Trust ID™)
